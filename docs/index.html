DSB Law Group

Next.js website repository

This GitHub Pages site hosts project documentation for the DSB Law Group website codebase. The full application is a server-rendered Next.js app with MySQL, admin CMS, newsletter worker, and authentication.

Production deployment

  • Run with Docker Compose and MySQL on a VPS or cloud server.
  • Copy .env.example to .env and configure database, email, and Redis.
  • Run migrations and seed: npm run db:migrate, npm run db:seed.
  • Start app and newsletter worker per the repository README.

Why this is not the live public site

GitHub Pages serves static files only. It cannot run API routes, admin login, database queries, or newsletter processing. Use Docker/VPS hosting for the full site.
